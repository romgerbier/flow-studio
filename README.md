# 🎤 Flow Studio — Créateur de Parodie

Application mobile pour créer des parodies musicales en respectant le flow et les syllabes de la chanson originale.

## Fonctionnalités
- Analyse syllabique ligne par ligne
- Carte des temps forts / faibles
- Score de flow en temps réel
- Suggestions IA (Claude)
- Export prompt Suno/Udio

## Installation locale
```bash
npm install
npm run dev
```

## Déploiement Netlify
Connecte ce repo sur netlify.com → déploiement automatique.
